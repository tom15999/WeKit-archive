# 模块设置说明

模块设置底栏包含「主页」「功能」「日志」「设置」四个 Tab, 可点击底栏或左右滑动
切换。进入功能分类或开放源代码许可页面后, 使用左上角返回按钮或系统返回手势返回。

## 主页

主页汇总当前模块状态:

| 内容 | 说明 |
|------|------|
| 模块已激活 | 显示当前模块版本; 在 LSPosed 环境中点击会尝试打开 LSPosed 管理器 |
| 已启用功能 | 显示当前已开启的功能数量, 点击进入「功能」Tab |
| 全部功能 | 显示当前版本包含的功能总数, 点击进入「功能」Tab |
| 系统信息 | 显示微信版本、模块版本、构建时间、设备型号、Android 版本、当前加载器和 Hook 桥接 |

## 功能设置

「功能」Tab 顶部可按功能名称或说明搜索。搜索结果可直接开关功能; 清空搜索内容后
恢复分类列表。进入分类后, 普通功能显示为开关, 名称旁带齿轮图标的功能可点击进入
详细配置, 并排列在普通开关之后。

- **聊天** — 消息显示增强、消息阻止、输入工具栏等
- **联系人与群组** — 联系人管理、群组增强
- **红包与支付** — 自动抢红包、指纹支付等
- **朋友圈** — 朋友圈广告拦截、查询增强等
- **系统与隐私** — 系统级功能、隐私保护、环境伪装等
- **音视频通话** — 音视频通话界面和行为增强
- **通知** — 通知增强
- **界面美化** — 微信界面视觉定制
- **公众号** — 公众号浏览增强
- **小程序** — 小程序去广告、限制解除等
- **视频号** — 视频号下载、限制解除
- **个人资料** — 修改昵称、头像、签名
- **调试** — 崩溃拦截、日志、分析工具
- **脚本 (JS)** — JavaScript 脚本和触发器
- **脚本 (Java)** — Java / BeanShell 脚本和 Hook
- **娱乐** — 整蛊、娱乐功能
- **批量操作** — 对多项数据执行批量处理
- **首页右上角菜单** — 自定义微信首页右上角 `+` 菜单项
- **联系人详情页面** — 联系人详情页功能增强

> 各功能的详细说明请参见 [功能文档目录](SUMMARY.md)。

## 日志

「日志」Tab 可直接查看模块生成的日志:

| 操作 | 说明 |
|------|------|
| 运行日志 / 崩溃日志 | 在两类日志之间切换 |
| 选择日志文件 | 按文件名、大小和修改时间选择要查看的日志 |
| 下拉刷新 | 重新读取当前类型的日志文件 |
| 分享 | 通过系统分享菜单发送当前日志文件 |
| 保存 | 通过系统文件选择器保存当前日志文件的副本 |
| 刷新 | 从右上角菜单重新读取当前日志 |
| 转到顶部 / 转到底部 | 快速跳转当前日志内容 |
| 清空 | 删除当前类型的全部日志 |

较长的运行日志内容默认折叠, 可点击展开。日志正文支持选择文本。

## 通用设置

「设置」Tab 顶部显示当前微信账号的头像、昵称和微信 ID。

### 界面

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 主题模式 | 选项 | 选择跟随系统、浅色模式或深色模式 |
| 自定义颜色 | 开关 | 使用自定义调色板生成模块配色, 而非 Miuix 默认蓝 |
| 动态壁纸取色 | 开关 | 使用系统壁纸强调色作为种子颜色, 需要 Android 12 (SDK 31) 或更高版本 |
| 种子颜色 | 按钮 | 动态壁纸取色关闭时, 手动选择生成配色所用的颜色 |
| 调色板样式 | 选项 | 选择从种子颜色生成配色的风格 |
| 颜色规格 | 选项 | 选择 Material 3 (2021) 或 Expressive (2025); 部分调色板样式仅支持 Material 3 (2021) |
| 同时对微信生效 | 开关 | 将自定义配色应用到微信界面, 重启微信后生效 |

除「主题模式」和「自定义颜色」外, 其余配色选项仅在「自定义颜色」开启后显示。

### 调试

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 详细日志 | 开关 | 输出高频日志 ⚠️ 可能会暴露你的隐私信息 |
| 显示加载完成 Toast | 开关 | 全部功能加载完成后显示 Toast 提示 |
| 清理消息内容微信 ID 前缀时允许非标准 ID | 开关 | 允许处理不带 `wxid_` 前缀的微信 ID, 可能误伤消息原始内容 (实验性) |

### 兼容

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 禁用版本适配 | 开关 | 开启后不会弹出 DEX 查找对话框, 未适配功能将不会被加载 |
| 重置适配信息 | 按钮 | 清除全部 DEX 适配信息, 等待下次启动时重新适配 |
| 宿主热更新时重新适配 | 开关 | 微信热更新时重置 DEX 缓存并重新适配, 可能导致频繁重新适配 (实验性) |

### 配置

| 设置项 | 说明 |
|------|-------|
| 导出配置 | 将当前模块全部 KV 配置导出为 JSON 文件。导出的内容包含功能开关、通用设置及大部分功能配置, 但不包含模块其他运行数据 |
| 导入配置 | 从 JSON 文件导入配置数据, 并与当前配置合并; 若存在相同键值, 则会被覆盖 |
| 清除配置 | 清除全部模块 KV 配置数据。该操作不可逆, 执行前会弹出确认提示以避免误操作 |

### 更新

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 检查更新 | 按钮 | 立即检查模块是否有新版本; 发现新版本后显示确认对话框, 确认后下载并请求安装 |
